Day 5 – CSS Introduction

1. What is CSS?

CSS = Cascading Style Sheets. It controls the look and layout of HTML elements — colors, fonts, spacing, positioning, animations, and more.

HTML is the structure, CSS is the style.

2. How to Add CSS to HTML

There are 3 ways:

1. Inline CSS (in the HTML element)

2. Internal CSS

3. External CSS (link to a .css file)

CSS Syntax

A CSS rule has:

selector { property: value; }
