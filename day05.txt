Day 5 – CSS Introduction
1. What is CSS?
CSS = Cascading Style Sheets.
It controls the look and layout of HTML elements — colors, fonts, spacing, positioning, animations, and more.

HTML is the structure, CSS is the style.

Example:

html
Copy
Edit
<!-- HTML -->
<h1>Hello World</h1>

<!-- CSS -->
h1 {
  color: blue;
  font-size: 36px;
}
2. How to Add CSS to HTML
There are 3 ways:

1. Inline CSS (in the HTML element)
html
Copy
Edit
<p style="color: red;">This text is red.</p>
⚠️ Not recommended for big projects — hard to maintain.

2. Internal CSS (inside <style> in the <head>)
html
Copy
Edit
<head>
  <style>
    p {
      color: green;
    }
  </style>
</head>
3. External CSS (link to a .css file) ✅ Best Practice
html
Copy
Edit
<!-- HTML -->
<link rel="stylesheet" href="styles.css">

<!-- styles.css -->
p {
  color: purple;
}
3. CSS Syntax
A CSS rule has:

css
Copy
Edit
selector {
  property: value;
}
Example:

css
Copy
Edit
h1 {
  color: blue;         /* Property: color, Value: blue */
  font-size: 24px;     /* Property: font-size, Value: 24px */
}
4. Selectors
Selectors choose which HTML elements to style.

Basic Selectors
css
Copy
Edit
/* Element selector */
p { color: gray; }

/* ID selector */
#main-title { color: red; }

/* Class selector */
.highlight { background-color: yellow; }
